# 📝 Terminal To-Do

A lightweight Python CLI app to manage a to-do list right in your terminal.

## ✨ Features
- Add tasks
- Mark tasks as done
- Delete tasks
- Saves tasks to `todo.json`

## 🛠 Requirements
- Python 3.x

## 🚀 Usage
```bash
python todo.py

Commands inside app:

    add Buy groceries

    done 1

    del 2
